# AutoClocking Frontend

Frontend informativo en SolidJS para visualizar el historial de marcajes automáticos expuesto por la Cloud Function `marcajes`.

## Variables

Copiar `.env.example` y ajustar si se quiere cambiar el endpoint:

```bash
VITE_MARCAJES_API_URL=https://marcajes-vg7vvkcauq-ue.a.run.app
```

## Comandos

```bash
npm install
npm run dev
npm run build
```
